== Frequently Asked Questions ==
= How many menu does Cocktail theme support? =
There is three Menu registered in this theme. i.e. Main Menu,  Side Menu and Add Social Icons Only

= Where can i get instruction about theme? = 
For detail instructions visit https://themefreesia.com/theme-instruction/cocktail/

= Is theme compatible with wpml? =
wpml-config.xml is used to support multilangual plugins such and wpml, polylang.

== Installation ==
1. Primary: == Installation ==
	- In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
	- Type in Cocktail in the search form and press the 'Enter' key on your keyboard.
	- Click on the 'Activate' button to use your new theme right away.
	- Go to https://themefreesia.com/theme-instruction/cocktail/ for a guide on how to customize this theme.
	- Navigate to Appearance > Customize in your admin panel and customize to taste.

2. Alternate: Unzip the template file (cocktail.zip) that you have downloaded. Via FTP, upload the whole folder (cocktail) to your server and place it inside the http://yourdomain.com/wp-content/themes/folder(cocktail). Do not change directory name. The template files should thus be here now: http://yourdomain.com/wp-content/themes/cocktail/index.php (for example).
	- Log into your WP admin panel and click on the Appearance -> Theme tab. Now click on the Cocktail theme to activate it.
	- Want to enjoy Cocktail feature. Go to  Dashboard -> Appearance -> Customize and play with Cocktail Options
	- The theme is offered free of charge. If someone asked money for it, someone just tricked you.
